# Looper.AI

**Expertise, engineered.** — The AI copilot for golf coaching and fitting.

## What is Looper.AI?

Looper.AI is an AI copilot that sits alongside the people who make golfers better — coaches, fitters, and the golfers themselves. It captures session-level data, applies calibrated intelligence, and recommends what to try next.

The practitioner stays in control. The AI handles measurement discipline, math, and memory.

## Three Experiences

| Persona | Route | Description |
|---------|-------|-------------|
| **Golfer** | `/golfer` | Mobile app — track improvement, review lessons, practice with purpose |
| **Coach** | `/coach` | Desktop copilot — capture sessions, diagnose limiting factors, see what works |
| **Fitter** | `/fitter` | Desktop copilot — fit with full context, AI-powered equipment recommendations |

### Additional Pages

- `/vision` — 90-second animated product walkthrough (sizzle reel)
- `/narrative` — Full narrative: thesis, problem, solution, flywheel, business model
- `/thesis` — Business case with market sizing and five-year model

## Tech Stack

- **React 19** + **TypeScript** + **Vite**
- **Tailwind CSS 4** with custom design tokens
- **Recharts** for data visualizations
- **Lucide React** for icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build    # TypeScript check + Vite production build
npm run preview  # Serve the production build locally
```
